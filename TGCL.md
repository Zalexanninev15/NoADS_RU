# NoADS_RU TGCL

> Проект устарел и более не будет развиваться, переоформил readme файл.

Рекомендуется получать материалы с большинства каналов в обход своего основного аккаунта:
- через браузер (в ссылке перед **username** канала необходимо вставить `s/`)
- через RSS (см. ниже)
- из других источников (некоторые авторы заботятся об удобстве для своих читателей создавая контент/зеркала на других платформах, например, Mastodon)

Чем мне можно заменить подписку на данные каналы? Короткий ответ - RSS.

## RSS

### Варианты RSS Hub:
- `https://rsshub.top/telegram/channel/channel_username`
- `https://rsshub.app/telegram/channel/channel_username`
- `https://rsshub.rssforever.com/telegram/channel/channel_username`
### Варианты RSS Bridge *:
- `https://rss-bridge.org/bridge01/?action=display&username=channel_username&bridge=TelegramBridge&format=Html`
- `https://rss.catgirls.asia/?action=display&username=channel_username&bridge=TelegramBridge&format=Html`

> [!NOTE]
> **channel_username** необходимо заменить на **username** канала.
> * - для RSS Bridge необходимо выбрать нужный формат после перехода по ссылке, ссылки не прямые RSS.
